# Twilio Birthday Bot 🎂

Automatic WhatsApp message sender using Python and Twilio API.

## 🚀 How to use:
1. **Install Library:** 
   Run this command in your terminal:
   `pip install twilio`

2. **Setup Credentials:**
   Get your `Account SID`, `Auth Token`, and `Twilio Phone Number` from your [Twilio Console](https://www.twilio.com/console).

3. **Update Script:**
   Open `main.py` and replace the placeholders with your actual Twilio credentials and your friend's phone number.

4. **Run:**
   Execute the script:
   `python main.py`
